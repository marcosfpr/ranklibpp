Detailed Description

A DataPoint is an object to be ranked (Like in RankLib)

A DataPoint represent a pair [item, query] extracted in svm-light format for learning to rank tasks:

"<LABEL> qid:<QUERY_ID> [<FEATURE_ID: FEATURE_VALUE>]"
